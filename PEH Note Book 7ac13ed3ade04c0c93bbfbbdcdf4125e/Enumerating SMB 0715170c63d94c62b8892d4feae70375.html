Enumerating SMB

What is SMB:

SMB stands for Server Message Block, which is a protocol used for file sharing, network browsing, printing services, and interprocess communication over a network. It is commonly used in Windows-based systems for sharing files and resources between computers.

****The protocol provides a way for computers to access shared folders, printers, and other network resources.

SMB has evolved over the years, and different versions have been released, including SMB1, SMB2, SMB3, and SMB3.1. Each version introduced improvements in terms of performance, security, and features. For example, SMB3 introduced features like encryption, improved performance for large file transfers, and better support for network resiliency.

It's important to note that SMB is not limited to Windows systems. Many other operating systems, including Linux and macOS

Overall, SMB plays a significant role in enabling file sharing and resource access in networks, particularly in Windows-based environments.

SMB ports are generally port numbers 139 and 445.
